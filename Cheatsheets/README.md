# ARM64 Assembly Cheatsheets

Quick reference guides for ARM64 (AArch64) assembly programming.

## Available Cheatsheets

### 01_ARM64_Opcodes.md
Complete opcode reference with examples:
- Data processing instructions (move, arithmetic, logical, shift)
- Memory instructions (load, store, addressing modes)
- Branch instructions (conditional, unconditional, indirect)
- Atomic instructions (load-exclusive, store-exclusive)
- System calls
- Memory barriers
- SIMD/NEON instructions
- Register conventions
- Common patterns and code snippets

### 02_ARM64_Instruction_Set.md
Comprehensive instruction set reference:
- Complete instruction categories
- Instruction tables with descriptions and examples
- Addressing modes
- Instruction encoding notes
- Performance optimization tips

### 03_ARM64_Syscalls.md
Linux system calls reference:
- Syscall convention
- Common syscalls (exit, write, read, open, close, lseek)
- Syscall numbers table
- File descriptors
- Open flags and lseek whence values
- Error handling patterns
- Complete examples
